# spotify-canvas

show your recently played tracks on spotify with a canvas.

## requirements

* NodeJS 16.x
* Yarn
* Vercel CLI
* Spotify client id, client secret, and refresh token

## install

```shell
yarn install 
```

## local development

```shell
yarn build && vercel dev
```

## deploy to Vercel

```shell
yarn build && vercel --prod
```
